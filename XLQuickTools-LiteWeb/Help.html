Quick Tools (Lite)

Welcome! This guide provides an overview of the available tools and what they do.

Formatting Tools

  • Trim and Clean – Removes any leading, trailing or extra spaces and non-printable characters from cells. Removes two or more consecutive spaces between characters.
  • Text Options – Includes:
    • Add Leading/Trailing Text
    • Convert to Uppercase
    • Convert to Lowercase
    • Convert to Propercase (Title Case)
    • Remove Letters [A-Z]
    • Remove Numbers [0-9]
    • Remove Special Characters [~!@#$%^&*]
  • Date/Text Converter – Converts date formats or dates as text between different regional styles. Easily switch between Excel and Text dates.

Empty Cells

  • Delete Empty Rows – Removes rows that contain no data.
  • Delete Empty Columns – Removes columns that contain no data.
  • Fill Down – Fills down values in empty cells based on the cell above.

Data Validation Tools

  • Check for Duplicates – Identifies duplicate values in a column. Inserts a count if they do exist.
  • Filter – The built-in Excel filter, placed on the task pane for easier access while using other tools.
  • Column Information – Provides details on the selected column. Unique, non-blank, blank and total rows.
  • Reset Column – Ensures the column's formatting and data handling are reset. Numbers stored as text will be recognized as numbers.
  • Find Missing Data – Find missing data between two ranges. Creates a report on missing data and you can set it to highlight cells.
  • Compare Worksheets – Compares two worksheets. Creates a report on differences and you can set it to highlight cells.

Delimiter Tools

  • Split Columns to Rows – Splits delimited columns into rows. It can handle columns with varying delimited lengths.
  • Selection to Clipboard – Copies cells to clipboard with a delimiter, optionally adding leading/trailing text. 1,2,3.. 'a','b','c'

Use the Main Menu to return to the primary tool selection screen at any time.
